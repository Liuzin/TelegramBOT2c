Что такое БОТы в Telegram?

  1. БОТы - это специальные аккаунты в Telegram, являющиеся интерфейсом для кода, запущенного на нашем сервере;
  2. разработан для автоматизированной обработки сообщений;
  3. пользователи взаимодействуют посредством комманд, а мы контролируем его используя HTTPS-запросы.
  4. Есть API (готовый набор функций и процедур, которыми разработчики делятся с нами);

Как его создать?

  1. Отправить наш запрос BotFather
  2. Имя БОТа может быть любым, а вот username должен обязательно оканчиваться на "bot"
  3. Получить уникальный токен БОТа
  4. Пройти базовую настройку БОТа

Методы getUpdates и setWebhook

Нужен сервер и подписанный SSL-сертификат (на организацию защищенного https соединения)

setWebhook Telegram сам пересылает для обработки все сообщения, адресованные БОТу
getUpdates Мы должны каждый промежуток времени спрашивать у Telegram обновление

Запросы

https://api.telegram.org/bot<уникальнейший_токен>/<метод>

В результате json, в котором хранятся все id (сообщения, чата и юзера), дата, текст сообщения и т.д.
